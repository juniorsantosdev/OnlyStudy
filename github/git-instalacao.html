Instalando o Git

Acesse o link https://git-scm.com/downloads
Em seguida clique no download correspondente do seus Sistema Operacional.



Escolha 32 ou 64 bits e clique para iniciar o download.



Inicie o arquivo que foi baixado, em sequida clique em next



Selecione onde deseja instalar o git e clique em Next



Verifique se Git Bash Here e Git GUI Here estão selecionados e clique em Next



Clique em Next



Deixe o Vim como editor padrão e clique em Next



Selecione Let Git decide e clique em Next



Selecione Git from the command line and also from 3rd-party software e clique em Next



Selecione Use bundled OpenSSH e clique em Next



Selecione Use the OpenSSL library e clique em Next



Se o sistema for Windows, selecione Checkout Windows-style, commit Unix-style line endings e clique em Next



Selecione Use MinTT(the default terminal of MSYS2) e clique em Next



Selecione Default(fast-forward or merge) e clique em Next



Selecione Git Credential Manager e clique em Next



Selecione Enable file system caching e Enable symbolic links em seguida clique em Next



Não selecionar nada e clicar em Install



Quando a instalação terminar clique em Finish
