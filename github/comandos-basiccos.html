dir

Serve para listar todos os arquivos no diretório



cd /

Independente do diretório que estiver com esse comando volta para o diretório raiz do computador



cd nome_do_diretório

Faz com que você entre no diretório desejado, desde que você esteja um nível abaixo do mesmo.



cd ..

Serve para voltar um nível de pasta



cls

Usado para limpar a tela do terminal



mkdir nome_do_diretorio

Serve para criar um diretório



del nome_diretório

Usado para apagar todos os arquivos existentes no diretório



rmdir nome_diretório /S /Q

Serve para apagar o diretório juntamente com os arquivos contidos nele
