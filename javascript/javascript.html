Comentário de linha

Comentário de bloco

Variáveis

É um espaço na memória do computado destinado a um dado que é alterado durante a execução do algoritmo.
Em JavaScript há duas formas de declarar variáveis:

Var

Comando usado para declarar uma variável:
var (nome da variável) = (valor da variável)



Let

Let é uma variável de escopo, ou seja, ela só vai existir naquele bloco de código apenas.
let (nome da variável) = (valor da variável)



Constante

Usada para armazenar uma informação que não poderá ser alterada durante toda a estrutura do código.
const (nome da variável) = (valor da variável)

Estruturas Condicionais

Condicional Simples

Se a condição for verdadeira, irá executar o bloco de instruções, caso não seja a instrução é ignorada.



Condicional Compostal

Se a condição for verdareira, irá executar o bloco de instruções, caso não seja irá executar outro bloco de instruções.



Condicional encadeana

Usado para quando há mais de duas opções de verificação.

Controle de fluxo ( switch case )

Usado quando se têm as opções pré definidas.
